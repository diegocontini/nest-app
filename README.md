## Description

NestJS project made for studies.

## Installation

```bash
$ npm install
```

## Running the app

```bash

# starting the container
$ docker-compose up -d

# development
$ npm run start

# watch mode

$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```


## Nest generate

```bash
# create module
$ nest generate resource product
```
